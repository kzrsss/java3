# ☕ Java Demo — Learn Java by Example

A structured, beginner-friendly Java project covering the **core concepts** of Object-Oriented Programming and essential Java features. Each topic is organized into its own package, making it easy to explore and learn step by step.

## 📁 Project Structure

```
src/
├── abstraction/        # Abstract classes and methods
├── Class_and_Object/   # Creating and using classes and objects
├── database/           # Java database connectivity (JDBC)
├── encapsulation/      # Access modifiers, getters & setters
├── exceptions/         # Exception handling (try/catch/finally, custom exceptions)
├── factory/            # Factory design pattern
├── file_handling/      # Reading and writing files in Java
├── inheritance/        # Single, multilevel & hierarchical inheritance
├── interfaces/         # Interfaces and implementation
├── polymorphism/       # Method overloading & overriding
├── serialization/      # Object serialization & deserialization
├── singletone/         # Singleton design pattern
├── thread/             # Multithreading & concurrency
└── Main.java           # Entry point
```

## 🚀 Getting Started

### Prerequisites

- Java JDK 8 or higher
- Any IDE (IntelliJ IDEA, Eclipse, VS Code) or terminal

### Clone the Repository

```bash
git clone https://github.com/dilushamadushan/java-demo.git
cd java-demo
```
